## 使用方法

Claude が Trade History コメント依頼時に、このファイルを参照します。

### 参照ルール
- **理論的背景** が必要な場合 → #books を優先参照
- **実践的ノウハウ** が必要な場合 → #movies を優先参照
- **心理学的視点** が必要な場合 → #mentalhacks を優先参照

### Priority の定義
- ★★★ - 頻繁に参照。トレードの基本ルール
- ★★ - 時々参照。補足的な知見
- ★ - 参考程度。応用的な内容

### タグの意味
- **#books** - 書籍から抽出した理論的背景
- **#movies** - 動画から得た実践的ノウハウ
- **#oikawafx** - 及川式FX手法に関連
- **#mentalhacks** - トレード心理学・メンタル管理
- **#upper-timeframe-ma** - 上位足MA活用法
